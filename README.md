# Simple Portfolio Website

This project is a basic personal portfolio website created for a student/fresher.

It includes the following sections:

- Home Page
- About Section
- Skills Section
- Projects Section
- Contact Form

The website is fully responsive for both mobile and desktop devices with a clean and simple design. Basic navigation and smooth section transitions are included for better user experience.

Contact form submissions are stored in the database as required in the assignment.

## Run Locally

### Prerequisites

Make sure Node.js is installed on your system.

## Installation Steps

1. Clone the repository or download the source code.

2. Install all required dependencies:

`npm install`

3. Create a `.env.local` file in the root folder and add your required environment variables.

Example:

`GEMINI_API_KEY=your_api_key_here`

4. Start the development server:

`npm run dev`

5. Open the project in your browser after the server starts successfully.
